# spendly
Daily Expense Tracker - spendly.

## Development Approach

This project was built using a "vibe coding" approach. Instead of writing every line of code manually from scratch, I used AI tools to assist with generating parts of the code and structure.

My focus was on understanding the logic, designing the features, and guiding the implementation rather than typing every line myself.

This approach helped me move faster, experiment with ideas, and focus more on problem solving and building practical tools.
